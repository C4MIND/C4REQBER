How C4 Works

Every problem is classified into one of 27 cognitive states in a Z₃³ cube. The C4 engine then finds the optimal sequence of operators to navigate from the problem state to the discovery state.

Each state encodes three dimensions: Temporal (past/present/future), Scale (micro/meso/macro), and Abstraction (concrete/symbolic/meta).

T: Temporal Shift
T_INV: Reverse Temporal
S: Scale Shift
S_INV: Reverse Scale
A: Abstraction Shift
A_INV: Reverse Abstract
