# Мета-модель трансформаций домена «Религиоведение» 

Субдомены: сравнительное религиоведение, история религий, мистика, новые религиозные движения, феноменология религии, философия религии, психология религии, ритуальные исследования, священные тексты, социология религии. Структура: консолидация → кластеризация → абстрагирование → иерархия → мета-принципы → **операционная алгебра**.

## 0) Контекст и охват

- Источник: 10 CSV по субдоменам (∑ 150 процессов); после дедупликации — **150 уникальных**.
- Топ-объекты трансформации: отношения (12.7%), нормы (8.7%), поведение (8%), дискурс (7.3%), состояние (7.3%), содержание (6.7%), память/наследие (6%).
- Обратимость: **обратимые 67.3% / условно 10.7% / необратимые 22%**.
- Режимы: индивидуальные/постепенные, коллективные/постепенные, институциональные.

## ФАЗА 3: Абстрагирование (по кластерам)

### A) Аналитические

- **Мета-паттерн:** Дистанцированное изучение религиозных феноменов через категоризацию
- **Инварианты:** Методологическая нейтральность, систематичность
- **Вариативность:** Уровень эмпатии (от холодного анализа до вчувствования)
- **Открытия:** Компаративистика выявляет универсальные архетипы священного

### B) Трансформационные

- **Мета-паттерн:** Радикальное изменение религиозной идентичности/статуса
- **Инварианты:** Лиминальность, символическая смерть-возрождение
- **Вариативность:** Скорость (мгновенная конверсия vs постепенное обращение)
- **Открытия:** Необратимость многих трансформаций (22%)

### C) Регулятивные

- **Мета-паттерн:** Стабилизация религиозной системы через нормы и институты
- **Инварианты:** Авторитет, традиция, преемственность
- **Вариативность:** Жёсткость контроля (от мягкого наставничества до инквизиции)
- **Открытия:** Институциональные процессы чаще необратимы

### D) Интегративные

- **Мета-паттерн:** Объединение разнородного в новое религиозное целое
- **Инварианты:** Поиск общего, снятие противоречий
- **Вариативность:** Глубина синтеза (от поверхностного эклектизма до органичного слияния)
- **Открытия:** Синкретизм — ключевой механизм новых религиозных движений

### E) Коммуникативные

- **Мета-паттерн:** Передача священного знания/опыта между субъектами
- **Инварианты:** Медиация, символизм, перформативность
- **Вариативность:** Канал передачи (вербальный, визуальный, телесный, мистический)
- **Открытия:** Ритуал как универсальный коммуникативный медиум

## ФАЗА 4: Иерархия процессов

> **4-уровневая структура:**

- **Базовые (везде):** Интерпретация, классификация, ритуализация, передача традиции
- **Общие (в большинстве):** Сакрализация, институционализация, мифологизация, инициация
- **Специализированные (2–3 субдомена):** Экстатические техники, герменевтика, канонизация
- **Уникальные (один субдомен):** Эпохе (феноменология), глоссолалия (мистика), харизматическое лидерство (НРД)

## ФАЗА 5: Фундаментальные мета-принципы (7)

1. **Принцип священного различения** — отделение сакрального от профанного
2. **Принцип символической медиации** — любое религиозное содержание передаётся через символы
3. **Принцип ритуальной инвариантности** — воспроизведение архетипических действий
4. **Принцип мистической трансценденции** — выход за пределы обыденного сознания
5. **Принцип общинной интеграции** — религия как социальный клей
6. **Принцип традиционной преемственности** — передача через поколения
7. **Принцип адаптивного синкретизма** — включение нового без потери ядра
