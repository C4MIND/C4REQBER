## ФАЗА 3: Абстрагирование (по кластерам)

### A) Экспрессивные механизмы

- **Мета-паттерн:** Визуальное воплощение невидимых идей и эмоций
- **Инварианты:** Форма несёт смысл, эстетика коммуницирует, стиль = сообщение
- **Вариативность:** Степень абстракции, культурные коды, медиум
- **Открытия:** "Форма следует эмоции" как альтернатива "форма следует функции"

### B) Адаптивные механизмы

- **Мета-паттерн:** Подстройка решения под контекст использования
- **Инварианты:** Гибкость, модульность, респонсивность
- **Вариативность:** Глубина персонализации, автоматизация адаптации
- **Открытия:** Адаптивный дизайн как живая система

### C) Инновационные механизмы

- **Мета-паттерн:** Создание радикально нового через эксперимент
- **Инварианты:** Риск, прототипирование, итерация
- **Вариативность:** Степень радикальности, технологическая база
- **Открытия:** Спекулятивный дизайн как исследовательский метод

### D) Оптимизационные механизмы

- **Мета-паттерн:** Максимизация эффективности при минимизации усилий
- **Инварианты:** Простота, ясность, экономия
- **Вариативность:** Баланс простоты и функциональности
- **Открытия:** "Лучший дизайн — невидимый дизайн"

### E) Критические механизмы

- **Мета-паттерн:** Вопрошание и вызов существующим нормам
- **Инварианты:** Провокация, альтернатива, социальная ответственность
- **Вариативность:** Радикальность критики, конструктивность предложений
- **Открытия:** Дизайн как политическое высказывание

### F) Итеративные механизмы

- **Мета-паттерн:** Постепенное улучшение через циклы обратной связи
- **Инварианты:** Тестирование, обратная связь, инкрементальность
- **Вариативность:** Скорость итераций, методы валидации
- **Открытия:** Fail fast как принцип дизайн-мышления

## ФАЗА 4: Иерархия процессов

> **4-уровневая структура:**

- **Базовые (везде):** визуализация, композиция, итерация, прототипирование, стилизация
- **Общие (в большинстве):** брендинг, типографика, цветовое решение, пользовательское тестирование, адаптация
- **Специализированные (2-3 субдомена):** материаловедение (industrial/interior), информационная архитектура (ux/web), экологическое проектирование (landscape/industrial)
- **Уникальные (один субдомен):** дизайн-фикция (speculative), CMF-дизайн (industrial), пермакультура (landscape), микровзаимодействия (ux ui)

### 6.2. Плейбуки композиций (типовые маршруты)

> **Дизайн-цепочки:**
>
> | Сценарий             | Маршрут (микро-синтаксис)                                    | Цель                  |
> | -------------------- | ------------------------------------------------------------ | --------------------- |
> | Дизайн-мышление      | `Research → Define → Ideate → Prototype → Test → Iterate`    | Инновационное решение |
> | Брендинг             | `Research(market) → Conceptualize → Style → Compose → Test → Refine` | Идентичность бренда   |
> | UX-процесс           | `Research(users) → Define(problem) → Ideate → Prototype → Test → Optimize` | Пользовательский опыт |
> | Спекулятивный проект | `Research(future) → Speculate → Conceptualize → Visualize → Critique` | Провокация мышления   |
> | Продуктовый дизайн   | `Define → Sketch → Model → Prototype → Test → Manufacture`   | Готовый продукт       |
> | Веб-дизайн           | `Research → Wireframe → Design → Prototype → Code → Test → Launch` | Живой сайт            |

### 6.3. Покрытие (правило Парето)

- **Минимальный набор операторов**, покрывающий ≥80% сценариев: `Research, Ideate, Sketch, Prototype, Test, Iterate, Refine`
- **Редкие, но критичные:** `Speculate` (для дизайн-фикции), `Critique` (для социального дизайна), `Manufacture` (для перехода в производство).

## Инварианты домена

- **Итеративность абсолютна** — любое решение можно улучшить (100% обратимость!) 🔄
- **Форма несёт смысл** — визуальное = коммуникация 📣
- **Контекст определяет решение** — нет универсального дизайна 🌍
- **Пользователь в центре** — эмпатия как основа 👤
- **Простота сложна** — минимализм требует максимум усилий ⚡

## Параметры вариативности

- **Медиум** — физический/цифровой/гибридный/виртуальный 📱
- **Стиль** — минималистичный/экспрессивный/функциональный/декоративный 🎨
- **Масштаб** — микро (детали) / мезо (объекты) / макро (системы) 📏
- **Аудитория** — B2C/B2B/B2G/специализированная 👥
- **Темпоральность** — вневременной/трендовый/футуристический 🕐

## Практическая карта соответствия

1. **Начинающему дизайнеру** → освоить `Sketch → Iterate → Refine` для базовых навыков
2. **Продуктовому дизайнеру** → мастерство в `Research → Prototype → Test` для валидации
3. **Арт-директору** → фокус на `Conceptualize → Style → Compose` для визуального лидерства
