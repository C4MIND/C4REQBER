 **Режимы:** индивидуальные/дискретные практики с цикличностью и возможностью перехода в «фоновую» непрерывную осознанность; встречаются коллективные и ритуальные формы фиксации.

## ФАЗА 2: Кластеризация

### 2.1. Кластерная сетка (механизм × объект; компактная проекция)

- **Диагностические** (наблюдение, распознавание, метакинетика внимания)
   → над объектами *ум/сознание, эмоции, состояние*.
- **Регулятивные** (контроль темпа/ритма, дыхания, позы, «окна толерантности»)
   → над *состоянием, поведением, функциями*.
- **Трансформационные** (рефрейминг опыта, деконструкция «я», расширение поля внимания)
   → над *содержанием, значениями, структурой*.
- **Интегративные** (пронесение навыков в быт, ко-регуляция, сострадание, «микро-практики»)
   → над *отношениями, нормами, институтами*.
- **Коммуникативные** (вербализация, «labeling/note», диалог с терапевтом/наставником)
   → над *дискурсом, значениями, отношениями*.
- **Институциональные** (ритуалы, кодексы, расписание, сообщество/ретрит)
   → над *нормами, институтами, инфраструктурой*.

**Направления/полюса трансформаций:**
 *усиление ↔ ослабление* внимания/возбуждения; *связывание ↔ разрыв* автоматизмов; *локализация ↔ расширение* фокуса; *манифестация ↔ латентность* содержания; *централизация (якорь) ↔ децентрализация (растворение центра)*.

## ФАЗА 3: Абстрагирование

### A. Диагностические процессы

- **Мета-паттерн:** *Снятие неопределённости внутреннего опыта* (заметить → назвать → отпустить).
- **Инварианты:** наблюдение без слияния; «мета-позиция»; верификация через возврат к якорю.
- **Вариативность:** якоря (дыхание/тело/мантра/звук); зернистость лейблинга (общие метки ↔ тонкая феноменология); темп (микро-ноты ↔ длительная непрерывность).
- **Открытие:** чем точнее «схема наблюдения», тем мягче требуются регулятивные костыли для стабилизации.

### B. Регулятивные процессы

- **Мета-паттерн:** *Контур обратной связи «возбуждение–успокоение»* через дыхание, позу, темп, ритм и окно толерантности.
- **Инварианты:** петля «замечаю отклонение → мягко возвращаю → стабилизирую».
- **Вариативность:** счёт/удлинение выдоха, паузы, сканирование тела, настройка внешних условий.
- **Открытие:** регуляция — «несущая балка» для всех школ: без неё трансформация сорвается в реактивность.

### C. Трансформационные процессы

- **Мета-паттерн:** *Переконфигурация отношения к опыту* — от «я = содержание» к «я = пространство восприятия».
- **Инварианты:** децентрация/дефьюжн, терпимость к дискомфорту, неоценивание.
- **Вариативность:** рефрейминг (MBCT), исследование импермантности (Випассана), «скольжение по мантре» (ТМ).
- **Открытие:** устойчивые сдвиги происходят, когда переключения становятся **микро-привычками** в быту.

### D. Интегративные процессы

- **Мета-паттерн:** *Сборка навыка в повседневность* (уроки «off-cushion»).
- **Инварианты:** перенос в коммуникацию, труд, сон; сострадание/доброжелательность как связующее.
- **Вариативность:** одиночная/парная/групповая практика; микро-паузы; «тихие» напоминатели.
- **Открытие:** сочленение с отношениями радикально снижает откаты после ретритов/курсов.

### E. Коммуникативные процессы

- **Мета-паттерн:** *Переупаковка переживаний в понятные метки и инструкции*.
- **Инварианты:** короткие, точные, доброжелательные формулы; проверка понимания.
- **Вариативность:** «noting»/«labeling», дневники, супервизии, телесная обратная связь.
- **Открытие:** язык — это «операционная система» внимания; плохие метки создают скрытые ловушки.

### F. Институциональные процессы

- **Мета-паттерн:** *Фиксация изменений* через ритуал, время, сообщество и кодекс безопасности/этики.
- **Инварианты:** расписание, границы интенсивности, наставничество/реферальность.
- **Вариативность:** домашние протоколы, ретриты, онлайн-форматы, совместная практика.
- **Открытие:** институты нужны не ради формы, а для **перенастройки привычек среды** (триггеры/поддержка/нормы).

## ФАЗА 4: Иерархия процессов

### 4.1. Базовые (встречаются повсеместно)

*Якорение внимания (дыхание/тело) → замечание отвлечения → мягкий возврат → неоценивание → доброжелательность → фиксация в рутине (малые дозы, часто).*

### 4.2. Общие (в большинстве субдоменов)

*Сканирование тела, метки/labeling, регулирование темпа/ритма, работа с триггерами, микро-паузы, дневники/рефлексия, ко-регуляция в группе, этика безопасности.*

### 4.3. Специализированные (2–3 субдомена)

*Когнитивная дефузия/децентрация (MBCT↔Випассана), сострадание/метта (MBSR↔Випассана), протоколы «якорь-мантра» (ТМ↔дыхательная), поведенческие «экспозиции» к внутренним состояниям (MBCT↔MBSR).*

### 4.4. Уникальные (узкие подписи школ)

*Непрерывное «noting» (Випассана), безусильное повторение мантры (ТМ), структурные «сессии стресса» (MBSR), когнитивные карты триггеров/убеждений (MBCT), точные протоколы дыхательных счётов (дыхательная осознанность).*

## ФАЗА 5: Фундаментальные мета-принципы (5–10)

1. **Внимание — ограниченный ресурс и главный рычаг.** Якорение/возврат формируют «канал» для всех прочих изменений.
2. **Мета-осознанность > содержимое.** Сдвиг от слияния к наблюдению делает реакции опциональными.
3. **Невовлечённость + доброжелательность = безопасная пластичность.** Отсутствие оценки и теплоту стоит считать «условием эксперимента».
4. **Ритм и телесность управляют «окном толерантности».** Дыхание/поза/темп — быстрые регуляторы состояния.
5. **Микро-привычки побеждают «большие подвиги».** Частые малые циклы важнее редких длинных сессий.
6. **Язык организует внимание.** Короткие точные метки и инструкции — часть технологии, а не «объяснялки».
7. **Этика — не надстройка, а каркас.** Без безопасных границ и намерения трансформация нестабильна.
8. **Интеграция в отношения закрепляет сдвиг.** Ко-регуляция и сострадание уменьшают откаты и «эффект ретрита, потерянный в быту».
9. **Условная обратимость — правило.** Изменения держатся, пока поддержан контекст (сон, питание, среда, практика).
10. **Итеративность и мягкость.** «Замечай → возвращай → отпускай» — универсальный цикл для любых гранулярностей практики.

## Операционная «алгебра» домена (канонические ходы)

**Anchor / Notice / Label / Return / Breathe / Pace / Soften / Relax / Ground / Scan / Expand / Narrow / Defuse / Decenter / Allow / Accept / Investigate / Non-React / Reframe / Release / Count / Flow / Intend / Commit / Compassion / Co-regulate / Integrate / Ritualize / Journal / Iterate.**
 Типовые композиции:

- **Anchor(дыхание) → Notice(отвлечение) → Label(«мысль/звуки/тело») → Return → Allow/Non-React → Iterate.**
- **Scan(тело) → Pace/Soften → Expand(поле внимания) → Defuse/Decenter → Reframe(опыт) → Integrate(микро-паузы в быту).**
- **Intend(этика/намерение) → Practice(протокол) → Compassion(к себе/другим) → Co-regulate(группа) → Ritualize(расписание) → Journal/Reflect.**

------

## Инварианты домена

- **Безопасность и доброжелательность** как фон; **петля внимания** (заметил → вернулся); **телесный регулятор** (дыхание/поза); **язык как интерфейс**; **итеративность** и **микро-шаги**.

## Параметры вариативности

- Якорь (дыхание/тело/мантра/звук), ширина фокуса (узко ↔ широко), темп/ритм, индивидуальная ↔ групповая практика, глубина меток (грубые ↔ тонкие), формат (дом/ретрит/онлайн), длительность (секунды ↔ часы).

------

## Практическая карта соответствия (shortlist)

1. **Анти-стресс шлюз:** *Anchor → Pace/Breathe → Soften → Non-React → Return* (3–5 циклов по 30–60 с).
2. **«Off-cushion» интеграция:** *Intend утром → 5×1-мин микро-паузы днём → Journal вечером.*
3. **Работа с триггером:** *Notice триггер → Label(«мысль/тело/эмоция») → Ground(стопы/дыхание) → Allow → Reframe → Integrate (в коммуникацию).*

------

## TL;DR

Медитация — домен **условно-обратимых** сдвигов внимания/состояния, где устойчивость достигается *якорением, регуляцией ритма и этическим контейнером*. Шесть семейств процессов (диагностика, регуляция, трансформация, интеграция, коммуникация, институционализация) действуют над умом/телом/отношениями/нормами. «Алгебра» (*Anchor→Notice→Label→Return→Allow…*) задаёт маршруты от реактивности к устойчивому, тёплому присутствию в повседневности.
