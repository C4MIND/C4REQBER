## 6. РЕДУКЦИЯ: Почему 153 → 27 Сохраняет Мощность

### 6.1 Механизм редукции

**Шаг 1: Устранение синонимов**
- "Прошлое", "Прошлое 2", "Прошлое 3", "Ретроспектива" → T=0
- "Детали", "Конкретное", "Фокусированное" → D=0
- "Я", "Самооценка", "Внутренний контроль" → A=0

**Шаг 2: Выявление производных**
- "Планирование" = Future + Abstract = (2, 1, *)
- "Эмпатия" = Present + Concrete + Other = (1, 0, 1)
- "Системное мышление" = * + Meta + System = (*, 2, 2)

**Шаг 3: Факторизация суперкатегорий**
- "Мотивация" = функция от T (к чему стремится)
- "Риск" = функция от T (неопределенность будущего) + D (оценка)
- "Социум" = проекция A-оси

### 6.3 Почему именно 3 значения на ось?

**Аргументы из материалов:**

1. **Эмпирическая конвергенция:** Во всех моделях время делится на 3 (Past/Present/Future), а не на 2 или 4.

2. **Когнитивная экономия:** 3 — минимум для описания направления (−/0/+) или иерархии (низ/сред/верх).

3. **Логические типы Рассела:** Self < Other < System образуют иерархию типов.

4. **Психологическая реальность:** Исследования показывают, что люди спонтанно категоризируют в 3-5 групп (Miller's 7±2 → ядро ~3).

## 8. СВЯЗЬ С FRA (Fractal Routing Algorithm)

### 8.1 От статики к динамике

**C4** описывает *где* находится когнитивный агент.
**FRA** описывает *как* маршрутизировать задачу к оптимальному обработчику.

```
Задача → FRA определяет d_eff (эффективную размерность)
      → Маршрутизация к эксперту с matching capacity
      → Эксперт работает в своей области C4 пространства
```

### 8.2 Связь с Matrix Dream

**FRA** можно рассматривать как *автоматический выбор* QZRF операторов:

1. Задача высокой сложности (d_eff > 8) → Manifold Twist, Non-Local Shift
2. Задача средней сложности → Constructive Resonance, Fractal Zoom-In
3. Простая задача → Superposition Collapse, Resonance Pruning

### 8.3 Теорема адаптивной маршрутизации

**Утверждение:** Для любой когнитивной задачи существует оптимальная траектория в C4, реализуемая композицией QZRF операторов.

**Следствие:** FRA — это алгоритмическая реализация навигации в когнитивном пространстве, построенном из первичных метамоделей.
