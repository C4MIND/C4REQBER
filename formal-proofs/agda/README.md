# Agda Proof Directory

## Language

[Agda](https://agda.readthedocs.io/) — a dependently typed functional programming language for writing proofs.

## Verification

Type-check a single file:
```bash
agda 0000-template.agda
```

## Naming Convention

```
NNNN-theorem-name.agda
```

## Template Reference

See `0000-template.agda` for the expected file structure.
