# Coq Proof Directory

## Language

[Coq](https://coq.inria.fr/) — an interactive theorem prover based on the Calculus of Inductive Constructions.

## Verification

Compile a single file:
```bash
coqc 0000-template.v
```

## Naming Convention

```
NNNN-theorem-name.v
```

## Template Reference

See `0000-template.v` for the expected file structure.
