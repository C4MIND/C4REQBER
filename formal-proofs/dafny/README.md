# Dafny Verification Directory

## Language

[Dafny](https://dafny-lang.github.io/dafny/) — a verification-aware programming language with built-in specification constructs.

## Verification

Verify a single file:
```bash
dafny verify 0000-template.dfy
```

Run with compile:
```bash
dafny run 0000-template.dfy
```

## Naming Convention

```
NNNN-theorem-name.dfy
```

## Template Reference

See `0000-template.dfy` for the expected file structure.
