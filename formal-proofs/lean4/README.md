# Lean 4 Proof Directory

## Language

[Lean 4](https://lean-lang.org/) — a dependently typed theorem prover and programming language.

## Verification

Run a single file:
```bash
lean4 0000-template.lean
```

Or build the project with Lake:
```bash
lake build
```

## Naming Convention

```
NNNN-theorem-name.lean
```

## Template Reference

See `0000-template.lean` for the expected file structure.
