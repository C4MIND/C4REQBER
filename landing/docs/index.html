Common Questions

FAQ

What is c4reqber?

A cognitive exoskeleton for AI agents and human researchers. 38 simulation engine bridges, 47 knowledge sources, formal verification, Z₃³ topology, and O₀→O₁→O₂ observer — turning questions into gated research proposals.

How much does it cost?

Free and open-source (AGPL-3.0). You only pay for LLM API calls. A full pipeline run costs $0.002–$0.11 depending on tier.

Do I need a GPU?

No. GPU is optional. Apple Silicon (MLX) and NVIDIA CUDA accelerate simulation; the core pipeline runs on CPU.

What operating systems are supported?

macOS and Linux are fully tested. Windows may work under WSL2.
