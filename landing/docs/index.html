Common Questions

FAQ

What is c4reqber?

A cognitive exoskeleton for AI agents and human researchers. It combines 36 simulation engines, 43 knowledge sources, formal verification, a Z₃³ cognitive topology, O₀→O₁→O₂ meta-cognitive observer, and 23 core metaprograms to transform research questions into paradigm-shifting dissertations.

How much does it cost?

c4reqber is free and open-source (AGPL-3.0). You only pay for LLM API calls. A full pipeline run costs $0.002–$0.11 depending on the tier.

Do I need a GPU?

No. GPU is optional. Apple Silicon (MLX) and NVIDIA CUDA are supported for accelerated simulation, but the core pipeline runs entirely on CPU.

What operating systems are supported?

macOS and Linux are fully tested and supported. Windows is not tested but may work under WSL2.
