What is c4reqber?

In plain language — no jargon required.

1

What

A terminal-first research assistant. You ask a scientific question — c4reqber searches 51 literature sources, runs simulations, checks proofs, and writes a gated research proposal. Not a chatbot. A pipeline.

2

Install

One command. Python 3.11+. Bring your own API key (OpenRouter, Mistral, etc.). Optional GPU for heavy sims.

3

Use

blast solve "your question" for a one-shot run. blast turbo for paradigm proposals. blast tui for the visual cockpit. blast serve --mcp for AI agents.

First Principles

Agents Can Talk. But Can They Think?

AI agents have language. They lack cognition. c4reqber is the scientific body for the agentic mind.

❌

No Simulation

LLMs generate hypotheses but cannot test them. No physics engine, no statistical validation, no ground truth.

❌

No Verification

ChatGPT "proves" theorems by generating convincing text. Formal verification requires Lean4, Coq, Dafny.

❌

No Blind-Spot Detection

Agents cannot recognize their own observational biases or identify gaps in their reasoning. c4reqber uses O₀→O₁ observer shifts to surface blind spots automatically.

❌

No Scientific Rigor

Falsifiability, real citations, reproducibility — the foundation of science. Absent from most agent frameworks.

c4reqber solves all four. Formal verification + 38 simulation engine bridges + Z₃³ cognitive topology + real citations + O₀→O₁→O₂ meta-cognitive observer.
