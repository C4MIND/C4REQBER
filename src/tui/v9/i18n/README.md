# TUI v9 i18n — HY-MT1.5-1.8B + NLLB-200 Pipeline

## Status: v9.4.0 STABLE (2026-06-10)

- 7 languages (en, ru, zh, ja, de, ar, hi)
- 30 keys per language × 7 = 210 translations
- **0 cross-contamination issues** (verified by NLLB-200 cross-contamination checker)
- Translation model: **HY-MT1.5-1.8B (Tencent Hunyuan)** via MLX, local
- Cross-check model: **NLLB-200-distilled-600M (Meta)** for lang-id

## Pipeline

```
en.toml  (source)
   │
   │  translate_hymt.py
   │  HY-MT1.5-1.8B (Tencent, MLX 8bit ~1.8GB)
   │  + c4_science_terms.json glossary (68 terms)
   │  + brand-name preservation (DeepSeek, C4REQBER)
   │  + garbage detection (NEVER, prompt-echo, etc.)
   ▼
{ru,zh,ja,de,ar,hi}.toml
   │
   │  nllb_check.py
   │  NLLB-200 + script-range verification
   │  flags Cyrillic in ZH/JA, Arabic in DE/HI, etc.
   ▼
nllb_check report (210/210 OK)
   │
   │  regen_i18n.py
   │  TOML → Go map literal in i18n.go
   ▼
i18n.go (210 entries)
```

## Files

- `*.toml` — 7 source-of-truth translation files (re-generate from these)
- `i18n.go` — generated by `regen_i18n.py` from TOML (DO NOT EDIT)
- `i18n_test.go` — coverage tests (TestNoCrossContamination, TestTLangSwitch, etc.)
- `pipeline/translate_hymt.py` — HY-MT1.5-1.8B translation script
- `pipeline/nllb_check.py` — cross-contamination checker (NLLB + script range)
- `pipeline/regen_i18n.py` — TOML → i18n.go generator
- `pipeline/c4_science_terms.json` — 68 scientific terms (RU/ZH/JA/DE/AR/HI) used as glossary

## How to re-translate

1. Install MLX-LM and download model:
   ```bash
   hf download mlx-community/HY-MT1.5-1.8B-8bit --local-dir ~/.c4reqber/models/hy-mt
   ```
2. Run translation:
   ```bash
   python3 pipeline/translate_hymt.py \
     --src i18n/en.toml \
     --out-dir i18n/ \
     --glossary pipeline/c4_science_terms.json
   ```
3. Check for cross-contamination:
   ```bash
   python3 pipeline/nllb_check.py --i18n-dir i18n/ --output report.json
   ```
4. Regenerate i18n.go:
   ```bash
   python3 pipeline/regen_i18n.py i18n/ i18n.go
   ```
5. Build + test:
   ```bash
   go test -count=1 ./i18n/
   go build -o bin/c4tui-v9 ./cmd/c4tui-v9
   ```

## Quality Notes

HY-MT produces higher-quality translations than the previous OpenRouter-based pipeline:
- RU: "Выполняется" (running) — more natural than "ИДЁТ"
- ZH: "准备好迎接你的第一次探索" — more vivid
- JA: "初回の発見に向けて準備ができています" — natural
- DE: "Wissenserwerb" — proper scientific term (vs crude transliteration)
- AR: "قيد التشغيل" — natural
- HI: "पहली खोज के लिए तैयार" — natural

Proper noun preservation (DeepSeek, C4REQBER, lang codes) is enforced both via glossary
and post-processing garbage detection.
